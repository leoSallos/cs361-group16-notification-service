Required Software:
- Node.JS (minimum version v10.1, tested in v24.8)
- npm

0. Database
    - The database is in directory `./data`
    - If the directory does not exist, a new one will be created
1. Install Node Packages
    - in the directory run the command `npm install` to install packages
2. Run Server
    - run the command `npm start` to start the server
3. Run Test Program
    - run the command `node testApp/test.js` while the server is running
